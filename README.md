# caffronix-agent-skills

## 一句话安装

把下面这句话发给你的本地 agent：

```text
请从 https://github.com/Stargazer1492/caffronix-agent-skills 仓库按照 INSTALL.md 指引安装 {skill-name}。
```

把 `{skill-name}` 替换为要安装的技能名。

## 技能列表

- `ai-job-analyze`：抓取并分析字节、阿里、腾讯、美团公开招聘站点中的 AI 岗位，生成 HTML 或 PNG 报告。

详细安装规则见 [INSTALL.md](INSTALL.md)。
